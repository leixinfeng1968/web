# 跨设备文件上传功能说明

## 📋 功能概述
现在您的课件上传系统已经支持从其他设备（如手机、平板、其他电脑）上传文件到您的uploads文件夹。

## 🌐 访问地址
- **本地访问**（在当前电脑上）: http://localhost:8080
- **网络访问**（在其他设备上）: http://192.168.2.9:8080

## 📱 使用方法

### 在其他设备上访问：
1. 确保其他设备与您的电脑在同一个WiFi网络下
2. 在其他设备的浏览器中输入：http://192.168.2.9:8080
3. 选择要上传的HTML课件文件
4. 点击"上传文件"按钮
5. 文件将自动保存到您的uploads目录

### 同一台电脑上的使用方法：
1. 打开浏览器访问：http://localhost:8080
2. 选择要上传的HTML课件文件
3. 点击"上传文件"按钮
4. 文件将自动保存到uploads目录

## 📂 文件保存位置
所有上传的文件都会保存到以下目录：
```
G:\初中数学动态课件\web-master\uploads\
```

## 🔒 安全说明
- 只有在同一网络下的设备才能访问
- 服务器运行时会显示访问地址信息
- 可以随时按 Ctrl+C 停止服务器

## ⚠️ 注意事项
1. **网络要求**：其他设备必须与您的电脑在同一个WiFi网络下
2. **防火墙**：确保Windows防火墙允许8080端口的访问
3. **文件类型**：目前只支持HTML格式的课件文件
4. **服务器状态**：服务器启动后，其他设备才能访问

## 🚀 启动服务器
在命令提示符或终端中运行：
```bash
python upload_server.py
```
